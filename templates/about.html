About ExpiredDomain.dev

Understanding how we track dropped domains

Data Source

ExpiredDomain.dev uses zone files from ICANN's Centralized Zone Data Service (CZDS). These zone files contain the complete list of registered domains for each tracked top-level domain (TLD).

What Are Dropped Domains?

A "dropped domain" is a domain that was present in yesterday's zone file but is missing from today's zone file. This typically indicates that the domain registration has expired or been deleted.

By comparing zone files day by day, we can identify which domains have been dropped, providing valuable insights for domain monitoring and research purposes.
