About ExpiredDomain.dev

Data Source

ExpiredDomain.dev uses zone files from ICANN's Centralized Zone Data Service (CZDS). These zone files contain the complete list of registered domains for each tracked top-level domain (TLD).

What Are Dropped Domains?

A "dropped domain" is a domain that was present in yesterday's zone file but is missing from today's zone file. This typically indicates that the domain registration has expired or been deleted.

By comparing zone files day by day, we can identify which domains have been dropped, providing valuable insights for domain monitoring and research purposes.

Disclaimer

This service is provided for research and monitoring purposes only. The fact that a domain appears as "dropped" does not guarantee that it is available for registration. Domains may be in grace periods, redemption periods, or may have been re-registered immediately after dropping.

Always verify domain availability through official registrars before attempting to register a domain.

How It Works

  1. Zone files are downloaded daily from ICANN CZDS (or manually placed in the data directory)
  2. Each day's zone file is parsed to extract all registered domains
  3. Domains from yesterday are compared with today's domains
  4. Domains that disappeared are marked as "dropped" and stored in the database
  5. The web interface allows you to browse, search, and filter these dropped domains
